# Visualisation-Project

## Usage
Setup a virtual environment
```sh
python3 -m venv venv
source venv/bin/activate
```

Windows (Command Prompt)
```sh
python -m venv venv
venv\Scripts\activate
```

Windows (PowerShell)
```powershell
python -m venv venv
Set-ExecutionPolicy -Scope CurrentUser -ExecutionPolicy RemoteSigned -Force  # Only if activation is blocked
.\venv\Scripts\Activate.ps1
```

Install the packages
```sh
pip install -r requirements.txt
```
